Declaración de Accesibilidad

Última actualización: 14 de abril de 2026

Nuestro Compromiso

En ThriveMom, nos comprometemos a hacer que nuestro sitio web y nuestra app sean accesibles para tantas personas como sea posible, incluyendo a personas con discapacidad. Diseñamos ThriveMom para que sea simple, de apoyo y útil en momentos reales de la vida, incluso cuando los usuarios se sienten abrumados, distraídos o sobrecargados cognitivamente. La accesibilidad está integrada en cómo diseñamos, probamos y mejoramos el producto.

Estado de Conformidad

Las Pautas de Accesibilidad para el Contenido Web (WCAG) definen los requisitos para mejorar la accesibilidad para personas con discapacidad.

ThriveMom cumple sustancialmente con WCAG 2.2 Nivel AA.

Continuamente auditamos y mejoramos la accesibilidad en el sitio web y la app. Nuestros esfuerzos actuales incluyen:

  • Contraste de color diseñado para cumplir o superar las proporciones WCAG AA (4.5:1 para texto estándar, 3:1 para texto grande)
  • Áreas táctiles diseñadas para cumplir o superar el mínimo recomendado de 44x44px
  • Indicadores de enfoque visibles para elementos interactivos y navegación asistiva
  • Interacciones operables con teclado en todo el sitio
  • Enlaces de salto y regiones de referencia adecuadas para usuarios de lectores de pantalla

Aunque buscamos el cumplimiento total, la accesibilidad es un proceso continuo y seguimos refinando y validando en distintos dispositivos y casos de uso.

Entornos Compatibles

ThriveMom está diseñado para funcionar con las herramientas de accesibilidad integradas en dispositivos modernos:

  • iOS: Lector de pantalla VoiceOver
  • Android: Lector de pantalla TalkBack
  • Escritorio: Lectores de pantalla NVDA, JAWS y VoiceOver
  • Cambio de tamaño dinámico de texto
  • Texto en negrita
  • Configuración de pantalla de alto contraste
  • Configuración de movimiento reducido

Usamos la fuente Lexend en toda la app para apoyar la legibilidad y reducir la fatiga visual.

Probamos los flujos clave del usuario, como unirse a la lista de espera, navegar por las preguntas frecuentes y enviar preguntas, usando estas herramientas.

Funciones de Accesibilidad y Trabajo Continuo

Interfaz de Chat (App)

La experiencia del chat está diseñada para funcionar con lectores de pantalla mediante regiones en vivo y gestión del enfoque. Los nuevos mensajes se anuncian cuando es posible, y el enfoque se dirige al contenido relevante.

Continuamos refinando este comportamiento, especialmente para conversaciones largas o de ritmo rápido.

Movimiento y Animaciones

ThriveMom respeta la configuración de movimiento reducido de tu dispositivo, minimizando o desactivando las animaciones automáticamente.

También hay disponible un control manual "Reducir Animaciones" en la configuración del Perfil de la app, para mayor control.

Contraste de Color

Diseñamos texto y elementos de interfaz para cumplir con los estándares de contraste WCAG AA tanto en modo claro como oscuro.

Auditamos periódicamente el contraste en las pantallas y seguimos mejorando la consistencia.

Tamaño de Áreas Táctiles

Los elementos interactivos están diseñados para cumplir con los tamaños mínimos recomendados de áreas táctiles, mejorando la usabilidad.

Continuamos validando las nuevas funciones para mantener la consistencia.

Si experimentas un problema que no esté listado aquí, por favor contáctanos. Tus comentarios influyen directamente en nuestras mejoras.

Enfoque de Evaluación

Evaluamos la accesibilidad mediante:

  • Pruebas manuales con VoiceOver y TalkBack
  • Pruebas automatizadas con axe DevTools y Lighthouse
  • Iteración continua basada en comentarios reales de usuarios
